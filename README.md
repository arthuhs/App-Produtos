# App de Cadastro de Produtos

Este é o frontend de um aplicativo móvel desenvolvido para a disciplina Desenvolvimento para Dispositivos Móveis.

## 🛠️ Tecnologias Utilizadas
* React Native
* Expo
* React Navigation

## ⚙️ Como rodar o projeto
1. Clone este repositório: `git clone https://github.com/arthuhs/App-Produtos.git`
2. Instale as dependências: `npm install`
3. Inicie o app: `npx expo start`

## 📱 Telas do Aplicativo
* Login
* Listagem de Produtos
* Cadastro de Produtos
